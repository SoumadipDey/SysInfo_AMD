# SysInfo for AMD GPUs
GUI System monitoring tool made using PySide2 . Shows CPU, GPU and RAM usage for systems with AMD GPU
## How to Use ? 
Steps:
  * 1. Execute the `OpenHardwareMonitor.exe` file within the `OpenHardwareMonitor` folder
  * 2. Run `main.py`
